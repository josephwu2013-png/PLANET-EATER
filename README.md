# PLANET-EATER
This is a planet eater game, which you eats planet, relaxing!
Local run:

Put index.html, main.js, and style.css in the same folder.
Serve that folder with any static server.
Open index.html through http://..., not just by double-clicking the file.
Examples:

python3 -m http.server 8000
Then open:

http://localhost:8000/
There is no install step and no build step.

For GitHub:

Push the repo.
Turn on GitHub Pages for the repository.
Set it to deploy from the branch/folder that contains index.html.
GitHub Pages will serve the game automatically.
